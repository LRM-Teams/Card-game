# 斗地主叙事像素风视觉规范 v1

> LRM-417 | 基准：caozs2 三张参考图（Eastward 级现代 16/32-bit 手绘像素）
> 状态：**待画风确认**（2026-07-23）

## 1. 风格定位

| 维度 | 定稿口径 |
|---|---|
| 风格名 | **Narrative Pixel（叙事像素）** |
| 对标 | 现代主机级 2D 像素艺术；高密度手绘、横向探索构图、电影化灯光 |
| 反例 | 8-bit 大颗粒、色块占位、CSS 渐变背景、SaaS 白卡片、矢量扁平、霓虹彩虹 |
| 验收 | **隐藏全部 UI 后，场景本身应像可商用的叙事像素游戏地图** |

## 2. 双线结构

| 线 | 场景 | 内容 |
|---|---|---|
| **A 室外·大厅** | 傍晚老街棋牌茶馆 | 高密度叙事背景；UI 融入旧电视/站牌/登记簿等道具 |
| **B 室内·对局** | 茶馆内牌桌 | 保留/升级 v2 台呢、牌面、按钮、HUD（Eastward 级细节密度） |

对标：欢乐斗地主信息层级与反馈强度；画风参考 Eastward 级叙事像素密度。**不抄袭竞品美术资产。**

## 3. 像素与画布

| 参数 | 值 | 说明 |
|---|---|---|
| 逻辑像素网格 | **1×1 native**（手绘逐像素，非缩放滤镜） |
| 主场景输出 | **1920×1080**（16:9） |
| 瓦片基准 | **16×16** 地面/墙面；**32×32** 中型道具；**64×64** 建筑模块 |
| 角色精灵 | **32×48** 行走帧（大厅远景可缩至 16×24） |
| 渲染 | `image-rendering: pixelated`；**禁止** blur / smooth |
| 色数 | 单精灵 ≤32 色；全场景主色板 48 色 |

## 4. 色板（低饱和复古）

详见 `docs/doudizhu-design-tokens.md` §15 `--ddz-np-*`。

## 5. 光照规范

| 规则 | 口径 |
|---|---|
| 主光源 | 右上方 45° 斜阳（傍晚） |
| 阴影 | 冷色投影（slate/purple），**非纯黑**；建筑投下长阴影 |
| 局部光 | 窗内暖光、灯笼、旧电视屏幕 — 橙黄 lamp 色，范围 ≤120px |
| 空气 | 远景 desaturate + 轻雾（有序抖动像素，非 gradient blur） |
| 暗角 | 四角 vignette，前景遮挡（灌木/围栏） |
| 层次 | **前景**（遮挡+暗）→ **中景**（活动区+UI）→ **远景**（山/天际） |

## 6. 构图规范

| 规则 | 口径 |
|---|---|
| 视角 | 轻微俯视侧视（~30°），横向卷轴可探索 |
| 主场景 | 老街棋牌茶馆外景 — 中央建筑为视觉锚点 |
| 人物比例 | 小（场景占 70%+ 画面） |
| 路线 | 石板路/泥路形成横向动线，引导视线至中央入口 |
| 密度 | 每 256×256 区域 ≥8 类可辨识物件（管线/招牌/植物/杂物） |

## 7. 字体

| 用途 | 字体 | 备注 |
|---|---|---|
| 场景招牌 | **自制像素字** 8×8 / 12×12 点阵 | 不直接用系统宋体 |
| UI 功能字 | `Press Start 2P` 或自制 6×10 中文点阵 | 保持像素边缘清晰 |
| 正文可读 | `ZCOOL KuaiLe` 仅用于长说明（≥14px），加 1px 暗描边 | 大厅次级文案 |

## 8. UI 融入世界

| 功能 | 世界化表现 | 状态 |
|---|---|---|
| 开始游戏 | 旧电视屏幕上的金色按钮 | default/hover/press/disabled |
| 房间码 | 车站站牌 / 木质告示板 | 输入框嵌木板 |
| 昵称头像 | 茶馆的客人登记簿 | 次级，不抢主场景 |
| 设置/音效 | 墙上电箱 / 开关面板 | 顶栏极简 |
| 匹配中 | 电视雪花 + 倒计时灯 | loading 态 |

**禁止**：白卡片、圆角 SaaS 面板、无材质矩形。

## 9. 空/错/加载态

| 态 | 表现 | 验收 |
|---|---|---|
| 加载 | 电视雪花 4 帧 + 灯笼微闪 | 不出现白屏/转圈 |
| 空房间 | 站牌空白 + 木质「暂无牌局」印章 | 仍保持场景叙事 |
| 错误 | `ui_error_stamp` 红章叠在告示板 | 文案 ≤12 字 |
| 断线 | 电线火花 2 帧 + 灰化 overlay 15% | 不弹 SaaS modal |

## 10. 动态细节（画风确认后）

| 元素 | 周期 | 幅度 |
|---|---|---|
| 灯笼 | 2.4s | 亮度 ±15% |
| 树叶 | 3.6s | 2px 摆动 |
| 电视雪花 | 匹配时 | 4 帧循环 |
| 远景车/人 | 8–12s | 横向慢移 |
| 灰尘粒子 | 5s | 3–5 个微粒上升 |

单段动画 ≤700ms；循环保持缓慢克制。
