# 邀请进房入口 / 弹层规范（LRM-385）

> Owner：小雅。接入：小林。对标 baseline：欢乐斗地主「分享邀请进房」体验（房间号/链接），**不做**好友列表/社交图。

## 1. 信息架构

| 场景 | 入口 | 行为 |
|---|---|---|
| 房间页（房主/已入房） | 主操作区上方 **「邀请好友」** 金色次主按钮（带图标） | 打开邀请弹层 |
| 邀请弹层 | 房间号（短码 + 完整 UUID 可复制）+ 分享链接 + 复制/系统分享 | 关闭后回房间等人 |
| 大厅（被邀请方） | 次入口「私房」区：输入房间号 / 粘贴链接自动解析 | 加入后与邀请方同房 |

**硬规则**：大厅仍只有一个最亮主 CTA「开始游戏」；邀请入口在房间内，不抢匹配权重。

## 3. 弹层布局（对照）

```
┌─────────────────────────────┐
│  邀请好友同桌            ✕  │
│  把房间号或链接发给好友      │
│  ┌─────────────────────┐   │
│  │  房间号  ABC123      │   │
│  │  （完整 UUID 小字）   │   │
│  └─────────────────────┘   │
│  [ 复制房间号 ] [ 复制链接 ] │
│  [ 系统分享 ]（可选）        │
└─────────────────────────────┘
```

- 宽：`min(400px, 92vw)`；移动端按钮纵向堆叠。
- 复制成功：按钮文案切换「已复制」1.2s（与现有 `room-code-card` 一致）。

## 4. 分享链接格式

`{origin}/?room={roomId}` — 与 LRM-173 私房深链一致；大厅 `Lobby` 自动 join。

## 5. 验收对照

| 步骤 | 预期 |
|---|---|
| 房主创建房间 | 房间页可见「邀请好友」 |
| 点邀请 → 复制链接 | 剪贴板含 `?room=` |
| 被邀请方打开链接 | 大厅自动加入 → 跳转 `/room` 同房 |
| 被邀请方手动输入房间号 | 同房，满 3 人可开局 |
